### 项目介绍
1.提供了在手机主页面显示当天待办事项的功能，通过添加桌面小部件来实现。<br>
2.没有提供准时提醒功能，但提供了起始日期和到期日期的设置，主要用于待办事项的规划。<br>
3.提供了导入导出功能，只要发送导出的文件，就可在不同的手机上使用。<br>
